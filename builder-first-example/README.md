# ハウストゥエンティーワン Website

A Japanese real estate website exported from Figma design.

## Files

- `index.html` - Main HTML file with semantic structure
- `styles.css` - Complete CSS styling with organized sections

## Features

- **Responsive Design**: Includes media queries for different screen sizes
- **CSS Variables**: Uses custom properties for consistent theming
- **Semantic HTML**: Proper HTML5 structure with meaningful tags
- **Clean CSS**: Organized by sections matching the HTML structure
- **Hover Effects**: Interactive buttons and links with smooth transitions

## Structure

1. **Header** - Sticky navigation with logo, menu, and contact buttons
2. **Hero Section** - Large banner with Japanese text and graphics
3. **Navigation Cards** - Quick access to different property types
4. **Property Categories** - Four main categories (Sale, Rental, Commercial, Parking)
5. **New Construction Event** - Featured event section
6. **LINE Official Account** - Social integration section
7. **Property Listings** - Grid layout for rental and sale properties
8. **Shamaison Properties** - Special property showcase
9. **Sidebar** - Fixed social media buttons and appraisal link
10. **Footer** - Company information, offices, and contact details
